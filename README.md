# SPER Allocation & Store Performance Dashboard

A flat, upload-driven Streamlit dashboard for analyzing SPER pivot CSV/Excel exports.

## Files
- `app.py` — Streamlit application
- `requirements.txt` — dependencies for Streamlit Cloud or local install
- `sample_sper_week14.csv` — sample file structure from the provided Week 14 export

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud
Upload all files in this folder directly to the root of a GitHub repository. Do not place them in nested folders.
